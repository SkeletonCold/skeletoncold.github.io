我是挺震惊的，当然也怪不了他，人家**嘛，可以理解
⬆保留了libraries，assets等可以联网下载的文件（你启动游戏，他自动会给你下载资源文件）【所以我的精简客户端就只保留了mod文件/版本文件，不含任何库文件以及初运行时解压的文件】

最后

什么垃圾服务器😅可快点死吧
再在最后说一下他们服务器名字的由来，是高性能服务器(High performance server)的简称，毫无技术含量，毫无意义！
如果你真的看完了上面讲的，那应该对他们的管理员对待服务器态度有一定了解了
啊，你这磨蹭磨蹭，你都摸不出东西来，你干啥呢？？（我也不知道我在说什么，词穷）
《总之就是非常的烂》
当然，也不是把我捧在道德的至高点，我们服也有缺陷，我是服主，通常来讲，在发现bug的第一时间会赶到现场查看emmm怎么说呢，我对服务器的付出，是远远大于他们一群人管理的服务器要好的，并且我们更专业（有幸认识一位大佬，会编程，会编曲，是个很厉害的人！）
我在3天之内，为1.16.5机械动力服务器用数据包添加了170+自定义进度，以引导玩家游玩mod包，其中包含一些梗，以在枯燥乏味的进度中添加一丝乐趣，目前（2021.2.5）我计划拍段视频，加速剪辑后向群友展现这个服务器

  • 对比
项 HPS 云之天地
客户端 模组包分开，且都是完整包，特别大，含有沉余文件 模组包合并，分精简包和完整包，仅包含版本jar，模组，服务器ip
服务端 开放时间未知，带宽未知（个人动态公网绑定域名）解决问题很慢，一般都会延时 24x7开放，36MBPS带宽，看到了都会去尝试帮助他，服主看得懂崩溃报告
“管理员”态度 怠慢，消极 积极
群内环境 就那么几个人说话，管理占一半，晚上21点后冷群 也是那么几个人说话，不过玩家之间的交流更多（都快成csgo交流群了）26小时高强度在线
